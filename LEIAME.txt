O objetivo inicial foi criar um site simples e que atendesse aos pedidos do cliente. O projeto foi concluído 
com todas as funcionalidades e atende 100% à necessidade do usuário. A aplicação está bem limpa e fácil de
navegar. Além de ter o total controle tanto no front-end quanto no back-end da inserção de dados por parte
do usuário, fazendo todas as validações necessárias com o Form Request do laravel.

Com a própria arquitetura do Laravel (MVC), fica bem mais simples deixar o código estruturado e organizado, fazendo
com que facilite as manutenções futuras.

Para Fazer o código funcionar , primeiramente deve-se criar um arquivo '.env' na raiz do projeto e adicionar o código a cima.
Logo após coloque o nome da base de dados criada no seu banco de dados local (voce deverá cria-la), na variável 'DB_DATABASE='.
Crie uma chave com o comando 'php artisan key:generate' na raiz do seu projeto pelo terminal prompt.

Após isso rode o comando 'php artisan migrate' na raiz do seu projeto pelo terminal prompt para adicionar as tabelas na base local,
que são necessárias para funcionamento da aplicação.

Feito isso, basta rodar o comando  php artisan serve, também na raiz do projeto pelo terminal prompt, abrir o link gerado 
no terminal, no seu navegador e adicionar '/orcamentos' ao final da url para ter acesso à pagina inicial da aplicação.
